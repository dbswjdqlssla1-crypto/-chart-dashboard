안드로이드 — 홈 화면에 추가

Chrome에서 이 페이지를 열고 오른쪽 위 점 3개 메뉴(⋮) → "홈 화면에 추가" 또는 "앱 설치"를 누르세요. 이름 확인 후 "추가"를 누르면 홈 화면·앱 서랍에 아이콘이 생기고, 탭하면 브라우저 주소창 없이 앱처럼 전체 화면으로 열려요. (삼성 인터넷 브라우저는 메뉴 → "현재 페이지 추가" → "홈 화면")
